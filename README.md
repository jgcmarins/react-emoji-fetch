# React Emoji Fetch
A react app that helps you find an emoji

## Live demo at
#### [http://joaomarins.com/react-emoji-fetch/](http://joaomarins.com/react-emoji-fetch/)

## Usage
#### Installation
```bash
npm install
```

#### Run
```bash
npm start
```
